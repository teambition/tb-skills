
# TB Skills

Teambition 技能集合仓库，为 OpenClaw 提供智能化的项目管理和团队协作能力。

## 📦 技能列表

### [dingtalk-teambition](dingtalk-teambition/)

Teambition 项目和任务管理技能，使用 User Token 认证，通过 TQL（查询语言）灵活查询和管理 Teambition 项目、任务、成员等。

**详细文档：** [dingtalk-teambition/README.md](dingtalk-teambition/README.md)

## 🚀 快速开始

### 环境要求

- Python 3.8+
- uv 包管理器

### 安装技能

```bash
# 使用 clawhub 安装
clawhub install dingtalk-teambition
```

### 配置认证

访问 [Teambition 开放平台](https://open.teambition.com/user-mcp) 获取 User Token，然后配置环境变量：

```bash
export TEAMBITION_USER_TOKEN="your_user_token_here"
```
